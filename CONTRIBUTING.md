# 贡献指南

感谢您对 RB-A-420-3 项目的关注！

## 如何贡献

### 报告Bug
1. 搜索是否已有类似问题
2. 创建Issue，描述问题环境、复现步骤
3. 附上相关代码/日志

### 提出新功能
1. 先搜索是否已有相关讨论
2. 创建Feature Request
3. 说明用例和期望实现

### 提交代码
1. Fork本项目
2. 创建功能分支 (`git checkout -b feature/xxx`)
3. 编写代码并测试
4. 提交Pull Request

## 开发环境
- PlatformIO
- ESP32开发板
- Arduino框架

## 代码规范
- 使用4空格缩进
- 添加必要的注释
- 保持代码简洁

## 问题反馈
如有问题请提交Issue
